### Analog Output
| #      | Name    |
| :----: | :-----: |
| 0 | HB temperature set |
| 1 | HC temperature set |
| 2 - 11 | E0-E9 temperature set |


### Analog Input
| # | Name  |
| - | ----- |
| 0 | HB current |
| 1 | HC current |

### Digital Output
| # | Name  |
| - | ----- |
| 0 | HB trigger |
| 1 | HC trigger |

### Digital Input
| # | Name  |
| - | ----- |
| 0 | HB temperature in range |
| 1 | HC temperature in range |
